# 🌍 Project 01 — World Happiness Report EDA
**Pluto Academy AI & ML Internship Program**

## Overview
Exploratory Data Analysis on the World Happiness Report 2019 dataset.  
Identifies global happiness patterns, regional differences, and key factors that drive national well-being.

## Dataset
[World Happiness Report — Kaggle](https://www.kaggle.com/datasets/unsdsn/world-happiness)  
File used: `2019.csv` (156 countries, 9 columns)

## Tech Stack
`Python` · `Pandas` · `NumPy` · `Matplotlib` · `Seaborn` · `Google Colab`

## What's Covered
- Data loading, inspection, and cleaning
- 5 EDA questions answered with code
- 7 visualizations (bar, histogram, scatter, heatmap, pie, line)
- 5 data-backed business/research insights

## How to Run
1. Open the notebook in Google Colab
2. Upload `2019.csv` from Kaggle
3. Run all cells top to bottom

## Key Insights
- GDP, Social Support, and Life Expectancy are the top 3 happiness drivers
- Western Europe averages ~7.0; Sub-Saharan Africa averages ~4.5
- Generosity has almost no correlation with national happiness (r ≈ 0.08)
